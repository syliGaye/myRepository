Description de la formation MCSA (Microsoft Certified Solutions Associate) SERVEUR

retrouver les sessions de formations

Windows 2012 R2 est un système d’exploitation serveur très déployé dans les infrastructures professionnelles, et toute réussite en tant qu’administrateur passe par sa maitrise.
il se compose de trois modules :

formation Installation et configuration de Windows Server 2012 (70-410)

formation Administrer Microsoft Windows Server 2012 R2 (70-411)

formation Installation et configuration de Windows Server 2012 (70-412)


La formation Installation et configuration de Windows Server 2012 (70-410) s’inscrit dans une série de formations Windows 2012 qui vous permettra d’être très opérationnel sur Windows 2012 R2 et aussi d’être un MCSE.

Pendant cette formation Windows Server 2012 R2 (70-410), vous allez tout d’accord voir comment installer et configurer un serveur Windows Server 2012 R2 de base avant de commencer à implémenter les différents services : services de domaines Active Directory 2012 R2, serveur DHCP, serveur DNS, serveur de fichiers et d’impression.

Aussi durant cette formation Windows Server 2012 R2 (70-410), vous allez comprendre et implémenter l’adressage IPv4 et IPv6 sous Windows 2012, et aussi apprendre comment automatiser des tâches avec Powershell.

La formation Administrer Microsoft Windows Server 2012 R2 est la deuxième d'une série de trois cours qui permettent aux participants d’acquérir les compétences et connaissances pour mettre en œuvre une infrastructure Windows Server 2012/Windows Server 2012 R2 dans un environnement existant.

Débutants sous Windows ou bénéficiant d’une première expérience, les administrateurs système trouveront dans cette formation Windows Server 2012 R2 (70-411), toutes les clés pour tirer pleinement parti de la flexibilité et de la polyvalence de cette nouvelle version, en particulier dans le domaine de la sécurisation des ressources.

Cette formation Windows Server 2012 R2 (70-411) couvre les tâches d'administration nécessaires pour maintenir une infrastructure Windows Server 2012 et Windows Server 2012 R2.

La formation Configuration avancée des services Windows Server 2012 (70-412) vous permettra d’acquérir les connaissances et compétences nécessaires pour effectuer la configuration avancée de Windows Server 2012 et Windows Server 2012 R2.

Pendant cette formation Windows Server 2012 (70-412), vous allez voir le clustering, stockage, AD CS, AD RMS, DAC, IPAM, Réplication, BranchCache…
