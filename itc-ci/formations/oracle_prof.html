Description de la formation ORACLE CERTIFIED PROFESSIONAL (OCP)

retrouver les sessions de formations

Le serveur Oracle Database 11g permet d'effectuer des transactions ultra-rapides, de prendre de meilleures décisions métier. Cela donne aux utilisateurs la possibilité de construire des applications de base de données efficaces et de haute qualité.

Grâce à cette formation Oracle Database 11g DBA 1 (1Z0-052), vous pouvez commencer votre chemin pour devenir l'indispensable DBA Oracle dans votre entreprise.

Cette formation Oracle Database 11g DBA 1 (1Z0-052) vous donne de solides bases pour exploiter Oracle 11g et construire des bases de données robustes.

Durant cette formation Oracle Database 11g DBA 1 (1Z0-052), Noureddine DRISSI vous apprend: l'installation, la gestion, l'architecture, et les composants d'une base de données Oracle, ainsi que les interactions entre les différents éléments.

Pendant cette formation Oracle Database 11g DBA 1 (1Z0-052), Noureddine vous montre comment créer une base opérationnelle, gérer correctement et efficacement les différentes structures, notamment via le contrôle des performances, sécuriser et gérer les utilisateurs, les techniques de sauvegarde et récupération, implémenter l'Audit Oracle.

À la fin de cette formation Oracle Database 11g DBA 1 (1Z0-052) vous serez en mesure de passer l'examen Oracle Certified Associate 1Z0-052 Oracle, une certification qui est presque obligatoire sur le marché du travail.


Objectifs

Décrire, comprendre l’architecture d’une instance et d’une base de données Oracle,,

Installer et configurer Oracle Database 11g,

Prise en main des outils d’administration avec Oracle Database 11g,

Configurer Oracle Net Services,

Gérer les structures de stockage de la base de données,

Surveiller et administrer les données d'annulation UNDO,

Gérer les données et les accès concurrents avec Oracle Database 11g,

Créer et gérer des comptes utilisateur sous Oracle Database 11g,

Réaliser les opérations élémentaires de sauvegarde/restauration,

Gérer les performances sous Oracle Database 11g.


Prérequis

Avoir suivi la formation Le langage SQL,

Connaissances de base du système d’exploitation Linux,

Expérience professionnelle des bases de données relationnelles,

Expérience en matière de conception de bases de donnèes.


Public concerné

Ingénieurs support,

Administrateurs DBA,

Développeurs Java,

Consultants techniques,

Toute personne désirant devenir indispensable DBA Oracle.
